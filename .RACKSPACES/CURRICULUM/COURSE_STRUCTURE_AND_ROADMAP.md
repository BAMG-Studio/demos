# Course Structure & Roadmap

Duration: 6–8 weeks (self-paced). Each module delivers runnable outcomes and evidence aligned to Rackspace requirements.

## Week 1–2
- Module 01: Secure AWS Foundation (IaC)
- Module 02: CI/CD Security Pipelines

## Week 3–4
- Module 03: AWS Security Tooling (IAM/KMS/GuardDuty/Config/CloudTrail/WAF)
- Module 04: Cloud-Native AppSec & OWASP Top 10
- Module 05: Compliance — NIST/PCI/SOX (automation-first)

## Week 5–6
- Module 06: CNAPP/CSPM (concepts + evaluation criteria)
- Module 07: Logging/Monitoring/SIEM
- Module 08: IR Automation (event-driven)

## Optional — Week 7–8
- Module 09: Container/Kubernetes Security
- Module 10: Portfolio & Interview Guide

Each module includes: objectives, prerequisites, deliverables, quickstart commands, verification, and interview talking points.
