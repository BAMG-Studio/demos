# 04 - Dashboards & Use Cases

## Dashboards
- Security Ops Overview: event volume, top services, errors.
- AuthN/AuthZ: console/API logins, failures, root use, MFA gaps.
- Network: accept/reject trends, top talkers, geo map.
- Threat Findings: GuardDuty severity/time, type, account.

## Use Cases
- Root login detection; CloudTrail tamper; IAM privilege change; SG 0.0.0.0/0; S3 public; DNS to new domains.

## Validation
- Generate sample findings and failed logins; ensure panels update.

Outcome: Analyst-ready views mapped to common threats.
