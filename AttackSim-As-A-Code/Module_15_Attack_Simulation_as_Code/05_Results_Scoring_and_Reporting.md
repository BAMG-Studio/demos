# 05 - Results, Scoring, Reporting

## Metrics
- MTTD, MTTR per scenario
- Coverage % (detected / total)
- Auto-remediated vs detected-only vs prevented

## Reporting
- Aggregate results into markdown/CSV; push to S3; notify via SNS/Slack.
- Trend over time; highlight regressions.

Outcome: Evidence of control effectiveness and improvements.
