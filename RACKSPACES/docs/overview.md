# Overview (Layman's Guide)

- EC2: cloud computers you rent.
- S3: file cabinet for data.
- IAM: security guard deciding who can do what.
- KMS: key vault that encrypts data.
- CloudTrail: security camera recording every API call.
- Config: auditor checking settings stay compliant.
- WAF: firewall in front of web apps.

## What this demo shows
- Secure-by-default infrastructure via Terraform.
- CI/CD security gates to catch issues early.
- Automation scripts for audits, incident response, and compliance.
- Clear documentation and journals to explain decisions and outcomes.
