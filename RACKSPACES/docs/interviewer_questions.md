# Questions for the Interviewer

- Role clarity: typical day, key responsibilities.
- Tooling: current CNAPP/CSPM, CI/CD, cloud platforms.
- Team: structure, collaboration with dev/ops.
- Challenges: biggest current security gaps.
- DevSecOps maturity: shift-left practices, policy gates.
- Growth: training, certifications, progression.
- Culture: proactive vs. reactive balance, staying current.
